# KitaKits - Financial Literacy App

A simulation-first financial literacy platform designed to help Filipino students practice real-world financial decisions through interactive scenarios before managing actual money.

> Note: This repository contains the **design, research, and product planning** for KitaKits.  
> This project was developed as part of a technopreneurship requirement and is being considered for further development in the future.  
> As such, detailed implementation code and full business strategies are not publicly disclosed.

---

## Project Overview

- **Goal:** Help students build financial confidence through safe, simulation-based learning  
- **Focus:** Experiential learning, decision-making, and long-term outcome visualization  
- **Output:** Figma prototype, market research synthesis, business model, and financial projections  

---

## Problem

Financial literacy education is mostly theoretical. Students are expected to manage budgeting, taxes, and investments without ever practicing real financial decisions, leading to confusion and fear of making costly mistakes.

---

## Solution

KitaKits introduces a simulation-based platform where users can experiment with financial decisions in a risk-free environment. By adjusting variables and visualizing long-term outcomes, users learn through experience rather than just instruction.

## Key Features

- Customizable financial simulations (budgeting, investing, taxes)  
- Timeline slider for long-term outcome visualization  
- Strategy comparison across financial decisions  
- Personal financial portfolios  
- Risk profile badges based on user behavior  
- Institutional dashboard for classroom integration  
- Freemium model with premium feature unlocks  

---

## Challenges

- Designing simulations that are realistic but still beginner-friendly  
- Balancing free and premium features without reducing user value  
- Simplifying complex financial concepts into intuitive UI/UX  
- Building a sustainable business model with limited startup capital  
- Aligning product features with real user needs through research
